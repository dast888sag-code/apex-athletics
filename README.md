# Apex Athletics — Интернет-магазин спортивного снаряжения

Премиальный, интерактивный одностраничный интернет-магазин (SPA) по продаже высокотехнологичного спортивного инвентаря.

Сайт разработан с использованием современного дизайна, адаптивной сетки, неоновых акцентов и встроенного интерактивного функционала:
- **Умная фильтрация и поиск:** Мгновенный поиск товаров по ключевым словам и фильтрация по категориям (Гаджеты, Обувь, Силовые, Аксессуары).
- **Интерактивная корзина:** Боковая панель корзины с подсчетом стоимости, изменением количества товаров и удалением.
- **Подробный просмотр (Quick View):** Модальное окно с детальными характеристиками каждого товара и увеличенным изображением.
- **Симуляция заказа:** Форма оформления заказа с выбором способа оплаты и красивым экраном успешного подтверждения покупки.
- **Темная и светлая темы:** Динамическое переключение с сохранением выбора в `localStorage`.

## Публикация сайта на GitHub и GitHub Pages

Вы можете сделать этот сайт общедоступным в интернете бесплатно с помощью сервиса **GitHub Pages**. Следуйте этим шагам:

### Шаг 1. Создайте репозиторий на GitHub
1. Перейдите на [github.com](https://github.com/) и войдите в свой аккаунт.
2. Нажмите кнопку **New** (Создать новый репозиторий).
3. Укажите имя репозитория, например: `apex-athletics`.
4. Оставьте его **Public** (публичным) и нажмите **Create repository** (не инициализируйте его файлами README или .gitignore, так как они уже созданы локально).

### Шаг 2. Инициализируйте Git локально и свяжите с GitHub
Откройте консоль/терминал (например, Git Bash, Command Prompt или PowerShell) в папке проекта `C:\Users\samsung\.gemini\antigravity\scratch\apex-athletics` и выполните команды:

```bash
# Инициализация локального репозитория (если еще не сделано)
git init

# Добавление всех файлов проекта в индекс Git
git add .

# Создание первого коммита
git commit -m "Initial commit: Apex Athletics storefront"

# Переименование ветки в main
git branch -M main

# Связывание локального репозитория с вашим удаленным репозиторием на GitHub
# (замените "ВАШЕ_ИМЯ_ПОЛЬЗОВАТЕЛЯ" на ваш логин на GitHub)
git remote add origin https://github.com/ВАШЕ_ИМЯ_ПОЛЬЗОВАТЕЛЯ/apex-athletics.git

# Отправка кода на GitHub
git push -u origin main
```

### Шаг 3. Включите деплой через GitHub Pages
1. Откройте страницу вашего созданного репозитория на GitHub.
2. Перейдите во вкладку **Settings** (Настройки) в верхнем меню репозитория.
3. В левом меню выберите раздел **Pages**.
4. В секции **Build and deployment** под заголовком **Source** выберите **Deploy from a branch**.
5. Ниже в выпадающем списке **Branch** выберите ветку `main` и папку `/ (root)`. Нажмите кнопку **Save**.
6. Подождите 1–2 минуты. GitHub соберет и опубликует ваш сайт.
7. Обновите страницу настроек. Вверху появится ссылка вида: `https://ВАШЕ_ИМЯ_ПОЛЬЗОВАТЕЛЯ.github.io/apex-athletics/`.

Поздравляем! Ваш премиальный сайт спортивного инвентаря будет доступен всему миру по этой ссылке!
